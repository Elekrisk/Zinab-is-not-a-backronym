# Zinab

*Zinab Is Not A Backronym*

Zinab is an interpreted,
statically typed programming
language.

## Types

Zinab contains multiple basic
types:

### Integer types

| Name | Description
|------|-------------
| i8   | 8-bit integer.
| i16  | 16-bit integer.
| i32  | 32-bit integer.
| i64  | 64-bit integer.
| u8   | Unsigned 8-bit integer.
| u16  | Unsigned 16-bit integer.
| u32  | Unsigned 32-bit integer.
